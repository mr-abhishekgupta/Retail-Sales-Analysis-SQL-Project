# 📊 Retail Operations Analytics (SQL)

## 🎯 Project Objective
This project analyzes a retail store's performance. It tracks sales trends and manages inventory using SQL to help the business make data-driven decisions.

## 🛠️ Key SQL Skills Used
- **DDL/DML:** Database design & data entry.
- **Joins:** Combining tables for revenue reports.
- **Case Statements:** Creating automated stock alerts.
- **Aggregations:** Calculating total sales & performance.
